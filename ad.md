🎉【正式上线】Visionary：基于 WebGPU · 面向下一代"大世界模型"的渲染引擎

🚀 核心亮点
Visionary 是首个基于 WebGPU 和 ONNX Runtime 的高性能 Web 3D 渲染平台，连接科研与应用的通用载体。
1. **ONNX 驱动**：浏览器直接跑 AI 模型，异构算法统一调度，万物皆可推。
2. **WebGPU 原生**：释放 GPU 极致算力，百万级粒子渲染无瓶颈、零延迟。
3. **Three.js 集成**：无缝接入 Web 图形生态，现有项目集成零门槛。

🎨 全格式支持
• 静态高斯：PLY, SPLAT, KSplat, SPZ, SOG
• 动态模型：支持 ONNX 导出的 4DGS、Neural Avatars
• 传统 3D：GLB, GLTF, FBX, OBJ

✨ 为什么选择 Visionary？
• **高度可扩展**：支持自定义算法导出为 ONNX，新算法（及未来算法）都能轻松在 Web 展示。
• **Web 原生**：无需安装，即点即用，浏览器内直接运行高性能 3D/4D 内容。
• **混合渲染**：完美融合传统 Mesh 与 Gaussian Splatting，统一光照与遮挡。
